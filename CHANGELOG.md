# 更新日志

## v0.0.2 (2026-04-18)

### 新增功能

- 支持通过 `-port` flag 自定义服务监听端口，默认 8080
- 新增哔哩哔哩视频解析
- 新增 Twitter/X 视频解析
- 新增微博图集解析
- 新增皮皮虾图集解析
- 新增小红书图集解析及 Live Photo 视频提取
- 新增抖音图集解析音频
- 新增前端页面批量图片下载功能
- 新增多种前端主题样式
- 新增 Basic Auth 认证支持（通过环境变量配置）
- 新增 pre-commit hooks
- 新增 Claude Code GitHub Actions 工作流

### 修复

- 修复 bilibili 解析器空错误处理导致的潜在 panic
- 修复快手 long-video 解析
- 修复快手解析问题 (#47)
- 修复抖音解析 ID 问题
- 修复皮皮虾视频解析
- 修复小红书部分图集有水印、不清晰问题 (#49)
- 优化小红书分享链接失效时的错误返回
- 修复批量下载图片在非魔法网络环境下因 JS 依赖无法加载的问题
- 修复最右解析
- 移除 jQuery 依赖，解决 CDN 加载问题

### 优化

- 升级 Go 版本至 1.22.7
- 升级依赖包版本：
  - `goquery` v1.8.0 → v1.9.3
  - `gin` v1.8.1 → v1.10.0
  - `resty` v2.11.0 → v2.15.2
  - `gjson` v1.14.1 → v1.17.3
- 小红书图集图片使用高清地址
- 修改 images 字段格式
- 移除 MCP 功能模块，精简项目
- 前端页面使用内嵌模板替代外部模板文件
